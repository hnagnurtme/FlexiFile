# FlexiFile - File Conversion and Management System

>FlexiFile is a web-based application designed to facilitate file uploads, conversions, and management. It provides functionalities for user authentication, file processing, and payment integration.

## Features

- **User Authentication:** Register and log in to access the system.
- **File Upload:** Upload various file types for conversion.
- **File Conversion:** Convert uploaded files to different formats (e.g., PDF to DOCX, DOCX to PDF).
- **Payment Integration:** Secure payment processing for premium features (e.g., VNPAY).
- **Cloudinary Integration:** Store and manage converted files on Cloudinary.
- **Asynchronous Processing:** File conversions are handled by a worker thread to ensure a responsive user experience.
- **User Account Management:** View file conversion history and manage account details.
